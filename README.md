### ReactJS-Redux starter

```
git clone https://github.com/plamen911/create-react-app-redux.git
cd create-react-app-redux
yarn
yarn start
```

OR

```
yarn create-react-app myapp
cd myapp
yarn add axios connected-react-router redux react-redux redux-devtools-extension redux-thunk history react-router react-router-dom prop-types
yarn start
```
